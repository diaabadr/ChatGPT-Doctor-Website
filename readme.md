### To run the Client:

- "create .env file"
- Add the env vars (check .env.example file)
- run command `npm install` or `yarn add`
- run command `yarn dev`

### To run the server
- "create .env file"
- Add the env vars (check .env.example file)
- run command `npm install` 
- run command `npm run start`
